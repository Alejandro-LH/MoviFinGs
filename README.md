# MoviFinGs

MoviFinGs es una app iOS de finanzas personales construida con `SwiftUI`, `SwiftData` y `Charts`.

El proyecto está enfocado en ayudar a visualizar movimientos, presupuesto y seguimiento mensual de una forma clara, rápida y centrada en el día a día.

## Estado del proyecto

Proyecto en desarrollo activo.

Actualmente ya incluye:

- Panel principal con saldo actual, ingresos del mes, gastos del mes y accesos rápidos.
- Registro de movimientos con soporte para ingresos, gastos, deudas, suscripciones, inversiones y ahorro.
- Filtros y búsqueda de movimientos por tipo, mes, apartado y subcategoría.
- Estadísticas mensuales con gráficos y resumen visual.
- Gestión de presupuesto mensual y seguimiento por secciones.
- Administración de subcategorías para organizar mejor los registros.
- Persistencia local con `SwiftData`.

## Stack

- `SwiftUI`
- `SwiftData`
- `Charts`
- Arquitectura basada en vistas, view models y servicios
